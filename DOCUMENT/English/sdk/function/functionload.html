Function Reference - Function load

This liberally function can load the WTD controlling function.
Those function can use liberally version only.

Format

    void WTD_FunctionLoad(WtdIL near *Wtd_Function);


Input

Input
WtdIL near *Wtd_FunctionStructure of WTD controlling function.



Output

Type
void nothing



Explanation

This command can load the WTD controlling function.
Librally version WTD have this function only.
About the controlling the WTD, you can call the fnctions via the structure of WTD controlling functions.
please see example.



Example.


#include "wtd.h"

/* global variable */
static	WtdIL	WtdIl;		/* WTD controlling functions */
static	Wtd_Sys	*WtdSys;		/* WTD system structure */
static	Wtd	*WtdState[20];	/* WTD part structure */

/* This is in a librally file */
extern	char	WTD_Work[4096];	/* WTD work area */
extern	void	WTD_FunctionLoad(WtdIL	near	*Wtd_Function);

/* main routine */
void main(){
	int	i;

	/* Get pointer of the WTD system structure */
	WtdSys = (Wtd_Sys*)(&WTD_Work + WTD_WorkAdd_System);

	/* Get pointer of the WTD part structure */
	for(i=0;i<20;i++){
		WtdState[i]=(Wtd*)(&WTD_Work + WTD_WorkAdd_Ch0 + i * WTD_WorkChSize);
	}

	/* Load the WTD controlling functions */
	WTD_FunctionLoad(&wtdIL);

	/* Stay and initialize of the WTD */
	wtdIL.Stay();

	/* Play the music */
	wtdIL.MusicPlay( far pointer of the music file );
