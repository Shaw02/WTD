Software Development Kit - .WSC file format

WSMAKE.COM can make .WSC (ROM image of the WonderSwan emulator) file.
This struct is the following table.

Struct

Address
0xE0000 to 0xEFFFF Music Data (.WTD or .FX file)
0xF0000 to 0xFFFEF Player and viewer (ROM_IMG.BIN)
0xFFFF0 to 0xFFFFF System area of the CPU i8086 and WonderSwan
Flow of the WSMAKE.COM is as follows.
  1. Load the .WTD or .FX file, and mapping it from memory address 0xE0000.
  2. file format check. (Is this music data of the WTD.)
  3. Load the "ROM_IMG.BIN" file, and mapping it from memory address 0xF0000.
  4. Write "EA 00 00 00 F0"(far jump instruction of the i8086) to memory address 0xFFFF0.
