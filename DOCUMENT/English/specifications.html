Specifications

Device Driver(Liberally)

    About the WTD, there are the liberaly version and the IL (In direct liberaly) version.
    User application can call the functions for controlling WTD.

    WTD made by microsoft macro assember version 6.11.

    User application do not use the following device because WTD use it.
    • Sound device.
    • Serial device.
    • H-blank device.
    As to the IL (In direct liberaly) version, WTD use heap area of 4096 bytes.
    It's heap area are in Wonder Witch system.
    Therefore, user can use only wonder witch system.

    As fot the liberaly version, other system can use it.
    WTD allot the memory of 4096byte in the DGROUP segment area.

Music/Effect data

    WTD can play music and effect.
    • 4 channels PCM of the WonderSwan.
    • 16 channels MIDI of the RS-232C.(38.4kbps)
    • MIDI channel can use 8 note(chord).
    • In the PCM channel of the WonderSwan, user can set 7bit value with volume and panpot.
      This is the some range as MIDI channel.
    • MML Compliler can calculate the check sum of the GS formats exclusive.

PCM Voice data (effect)

    WTD can play wave data.
    • Format = microsoft wave format.
    • Sampling rate = 12kHz
    • Quantum = 8bit (monaural)
