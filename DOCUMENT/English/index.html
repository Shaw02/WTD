Wonder Swan Total Sound Driver

This is device driver and tools for the music and effect of the wonder swan.
User can you WTD for the user wonder swan applications and games.

Tools

WTDSV.fx (WonderWitch Music Selector & Viewer for WTD)

    This is Wonder Witch (QUTE CO., LTD.) application.
    This is Music selector and viewer for the WonderWitch.
    WTDSV.fx can play .WTD (music) file with the WTD.fx.(In-Direct liberally)

WSWA.exe (WonderSwan Wave Audition for Windows)

    This is 32bit MS-Windows (Microsoft) application.
    This is PCM voice editor for the WonderSwan.
    WSWA.exe can make .WTV (voice) file.

WTC.com (MML Compiler)

    This is 16bit MS-DOS (Microsoft) application.
    This is compiler for the MML(Music Macro Language).
    WTC.com can make .WTD or .FX (music) file.
    MML Reference

WSMAKE.com (WonderSwan Emulator ROM image maker)

    This is 16bit MS-DOS (Microsoft) application.
    This is converter to make the rom image for the Wonder Swan Emulator.
    WSMAKE.com can make .WSC (ROM image for the WonderSwan Emulator) file.
    This ROM image have the music player and viewer.

DAT2ASM.com (Easy un assembler)

    This is 16bit MS-DOS (Microsoft) application.
    This is converter to make the .ASM (assembly language) file.
