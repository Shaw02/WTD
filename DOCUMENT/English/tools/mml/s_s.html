MML Reference - Sweep

Set the sweep of the WonderSwan PCM channel 3.

Format

    slevel,rate


Input

Input Range Default
level0 to 255 - Sweep level.
rate 0 to 31 - Sweep rate.



Explanation

This command can set the sweep of the WonderSwan PCM channel 3.
