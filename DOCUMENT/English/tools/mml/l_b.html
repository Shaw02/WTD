MML Reference - Pitch bend

Send the pitch bend change message.

Format

    Bpitch(, range)


Input

Input Range Default
pitch 0 to 16833 8192 Pitch bend
range 1 to 127 2(GM) Pitch Bend Sensitivity (Registered Parameter 0x00,0x00)



Explanation

This command can send the pitch bend change message and pitch bend sensitivity message.
About the Pitch Bend Sensitivity, it is Registered Parameter(0,0) of the control change message.
