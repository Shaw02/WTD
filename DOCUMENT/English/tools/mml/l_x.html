MML Reference - System exclusive message

Send the System exclusive message.

Format

    X{data[1],data[2],...,data[n],}
    X{data[1],data[2],...,(data[i],...,data[j]),...,data[n],}


Input

Input Range Default
data[n] 0 to 127 - system exclusive message without 240(0xF0:Start of Exclusive) and 247(0xF7:End of Exclusive)



Explanation

This command can send the system exclusive message.
If you surround data with (), compiler will calculate the check sum of the GS standard(Roland Co.,Ltd.), and compiler will supplement it after ().
