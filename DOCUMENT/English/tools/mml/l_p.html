MML Reference - Mode switch (PCM) / Dumper switch (MIDI)

PCM : Mode switch of the WonderSwan PCM.
MIDI : Send the Hold(Dumper pedal, Sustain) (Control Change No.64)

Format

    Ps


Input

Input Range Default
s 0 to 1 - 0:Disable / 1:Enable



Explanation

If setting channel is MIDI, this command can send the Hold(Dumper pedal, Sustain) of the Control Change No.64.
If setting channel is PCM, this command can set the following mode of the WonderSwan PCM.
Channel mode
0 to 15 Hold(Dumper pedal, Sustain) switch of the MIDI
128 This channel have not mode.
129 This channel have not mode.
130 Sweep mode of the WonderSwan PCM channel 3
131 Noise mode of the WonderSwan PCM channel 4
