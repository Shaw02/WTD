MML Reference - Gate time (Step)

Set the gate time by the step.

Format

    qq
    uu


Input

Input Range Default
q 0 to 65535 0 Keyoff time before note-off
u 0 to 65535 0 0:Disable
1 to 65535:Keyoff time after note-on.



Explanation

Those commands can set the gate time by the step.
