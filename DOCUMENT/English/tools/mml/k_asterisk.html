MML Reference - LFO switch

Set the switch of LFO(Low freqency oscillator)

Format

    *type,switch


Input

Input Range Default
type 0 to 2 - 0:Interval(Ptich Bend) / 1:Expression / 2:Panpot
switch 0 or 1 0 0:Disable / 1:Enable


Explanation

This command set the Disable/Enable switch of the LFO(low freqency oscillator).
