MML Reference - Macro

Define the macro.

Format

When you write those command, you must write in the beginning of a line.

$Sn MML


Input

Input Range Default
S a to z, A to Z - Macro number.
n 0 to 255 0
MML - - Strings of the Music Macro Language


Explanation

This command can define the macro data.
Nesting of the macro are still 8 times.
