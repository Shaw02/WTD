MML Reference - LFO(Low Frequency Oscillator).

Set the LFO(Low Frequency Oscillator).

Format

    mtype,dr,sp,lv,co


Input

Input Range Default
type 0 to 2 - 0:Interval(Ptich Bend) / 1:Expression / 2:Panpot
dr 0 to 255 0 Length of decay
sp 1 to 255 0 Speed of LFO.
lv -32768 to 32767 0 Amplitude of LFO.
co 1
2 to 255 		0 1 shot LFO.
Triangle LFO.



Explanation

This command can set the LFO(Low Frequency Oscillator).
