MML Reference - Include

This command can set the voice data for the WonderSwan Voice.

Format

When you write those command, you must write in the beginning of a line.

IN Filename



Input

Input Range Default
Filename - - File name of the including mml file.


Explanation

This command can include mml file.
A limitation size is 64kbyte.
