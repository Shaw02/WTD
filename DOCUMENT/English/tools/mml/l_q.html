MML Reference - Gate time (Ratio)

Set the gate time by the ratio.

Format

    Qq
    Uu


Input

Input Range Default
q 1 to 8 8 Gate time = ( q / 8 ) * 100 %
u 1 to 100 - Gate time = ( u / 100 ) * 100 %



Explanation

Those commands can set the gate time by the ratio.
Those commands are exclusive each.
