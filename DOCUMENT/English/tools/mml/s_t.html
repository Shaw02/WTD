MML Reference - Tempo

Set the tempo.
(Set the H-blank interrupt of the WonderSwan)

Format

    ttempo
    t%H-Blank


Input

Input Range Default
tempo 1 to 65535 120 (when timebase=48) Tempo.
H-Blank 1 to 65535 125 Count of H-blank timer.
H-Blank = ( 12000 * 60 ) / ( tempo * timebase )
1 step times = H-Blank / 12000 [sec]



Explanation

This command can set the tempo.(H-Blank timer.)
