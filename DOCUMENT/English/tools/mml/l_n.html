MML Reference - Non Registered Parameter.

Send the Non Registered Parameter. (Control Change No.6,98,99)

Format

    Nlsb,msb,data


Input

Input Range Default
lsb 0 to 127 127 Address of LSB
msb 0 to 127 127 Address of MSB
data 0 to 127 - Data



Explanation

This command can send the Non Registered Parameter. of the Control Change No.6,98,99.
