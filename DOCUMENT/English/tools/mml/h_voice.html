MML Reference - @

This command can set the voice data for the WonderSwan.

Format

When you write those command, you must write in the beginning of a line.

@ No, d0, d1, to ,d31
@F No, No2, Filename



Input

Input Range Default
No 0 to 15 - Setting voice number for the music data.
No2 0 to 15 - Reading voice number of the .WTV(voice) file.
d0 to 31 0 to 15 - Setting the wave(tone / voice) data of the WonderSwan.
Sampling rate = 32 / 1 wavelength.
Quantum number = 4bit
Filename - - File name of the .WTV(voice) file.


Explanation

This command can set voice data for the WonderSwan.
This is common data for the music and effect.
