MML Reference - Voice / Program change

Set the vocie / program change.

Format

    @no
    @Fno
    @Sno, decay
    @Rno


Input

Input Range Default
no 0 to 15(PCM)/127(MIDI) 255 Voice/Program number.(value 255 is disable.)
decay 0 to 65535 - Decay of 2nd voice.



Explanation

This command can set the voice.
Command
@ 1st voice/program number when key-on.
@F
@S 2nd voice/program number after decay.
@R 3rd vocie/program number after key-off.
