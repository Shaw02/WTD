MML Reference - Bank Select

Send the bank select. (Control Change No.0, 32)

Format

    Hlsb,msb


Input

Input Range Default
lsb 0 to 127 - lsb of bank number
msb 0 to 127 msb of bank number



Explanation

This command can send the bank number of the Control Change No.0, 32.
