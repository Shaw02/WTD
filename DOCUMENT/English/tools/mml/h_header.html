MML Reference - header

This is the header of the music file.

Format

When you write those command, you must write in the beginning of a line.

TI Title
FI Filename
TB Timebase
PA Part.
FX Switch



Input

Input Range Default
Title 24 chr - Title for the WonderWitch
Filename 16 chr - File name for the WonderWitch
Timebase 1 to 256 48 The number of the time base
Part 1 to 19 0 The number of the part.
Switch 0 to 1 0 If you want to make .FX file, you must set '1'.


Explanation

TB, PA commands can set header of the .WTD(music) file.
TI, FI, FX commands can set header of the .FX(WonderWitch format) file.
The time base is a number of tick for the quarter note.
