MML Reference - Accidental define.

Set the accidental.

Format

    {accidental,note}


Input

Input Range Default
accidental #, +, -, = = #,+ : Sharp
- : Flat
= : Natural
note c,d,e,f,g,a,b cdefgab Musical scale



Explanation

This command can set the accidental.
Command Note
c C Doh
d D Re
e E Mi
f F Fa
g G Sol
a A La
b B, H Si



Example.

Command Scale
{-be} B flat major, G minor
{-b} F major, D minor
{=cdefgab} (Default) C major, A minor
{#f} G major, E minor
{#fc} D major, B minor
