MML Reference - Modulation

Send the modulation. (Control Change No.7)

Format

    Mdecay, mod


Input

Input Range Default
decay 0 to 65535 0 Pitch bend
mod 0 to 127 0 Pitch Bend Sensitivity (Registered Parameter 0x00,0x00)



Explanation

This command can send the modulation of the Control Change No.1.
