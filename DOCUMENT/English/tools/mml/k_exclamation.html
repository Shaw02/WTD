MML Reference - Work switch

Set the work switch.

Format

    !


Input

No parameter.


Explanation

When you use this command, WTD read the value from the work memory in next command, and WTD set the parameter of the next command for its value.
As to the work address upper 8bit, " command can set it. (Default it 0x00)
As to the work address lowper 8bit, it is parameter of the command.

Offset address = ((" command parameter) x 256 + (using command parameter))
Segment address = work area of WTD.
