dat2asm (Easy un assembler)

Summary

This software is convert from the some file to the DB, DW and DD command of the MASM (Microsoft Macro Assembler).
This software is 16bit MS-DOS application.

Command line format

DAT2ASM option filename >output
    option
      option output format
      /B DB command.
      /W DW command.
      /D DD command.
    filename
      Convert source file.
    output
      output file.
