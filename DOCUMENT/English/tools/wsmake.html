wsmake (Wonder Swan Emulator ROM image maker)

Summary

This software can convert from the "*.WTD" and "*.FX" file to the ".WSC" file.
This software is 16bit MS-DOS application.
This software need the "ROM_IMG.BIN" file in the same folder as "WSMAKE.COM".
"ROM_IMG.BIN" is in the WTD(Music Driver), the player and viewer.

About the file
extension
*.MML MML(Music Macro Language) Sorce file.
*.WTD Music data for the WTD.
*.FX File format for the Wonder Witch(BANDAI and QUTE).
*.WSC ROM image file for the Wonder Swan Emulator.


Command line format

WSMAKE filename
    filename
      Convert sorcefile.( "*.WTD" or "*.FX" file )


How to use of the maked ROM-image

Key Work
A Play the music.
B Stop the music.
