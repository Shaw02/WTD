wtdsv (WonderWitch Music Selector & Viewer for WTD)

Summary

This software is music player and viewer of the WTD.
This software is WonderWitch application on the WonderSwan.
This software need the "WTDSV.FX" and "WTD.FX" or "WTD_H.FX" in the directory "/ROM0/" of the WonderWitch cartridge.

How to use

Key Work
X1 to X4 Select the music file.
Y1 and Y3 Change the view track.
Y2 and Y4 Change the view mode.
A Play the music.
B Stop the music.
Start Quit the wtdsv.
